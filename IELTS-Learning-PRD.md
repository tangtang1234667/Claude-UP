# IELTS Learning Website - PRD

## 1. 产品概述

一款面向雅思考生的在线学习网站，提供文章阅读跟读、单词记忆、智能练习三大核心功能，帮助用户高效备考雅思。

## 2. 目标用户

正在备考雅思的英语学习者，希望通过系统化工具提升听、读、词汇能力。

## 3. 核心功能模块

### 3.1 文章阅读模块

**功能描述：** 用户可导入感兴趣的英文文章，进行沉浸式阅读和跟读练习。

| 功能项 | 说明 |
|--------|------|
| 文章导入 | 支持两种方式：① 直接粘贴文本 ② 上传文件（.txt / .pdf / .docx） |
| 语音播放 | 文章支持 TTS 语音朗读，可调节语速（0.5x / 0.75x / 1.0x / 1.5x） |
| 跟读练习 | 逐句高亮播放，用户可点击任意句子单独播放并跟读 |
| 中文翻译 | 支持全文翻译和逐句对照翻译，可切换中英对照/纯英文模式 |
| 单词收藏 | 长按或点击文章中任意单词，弹出释义卡片，支持一键收藏至生词本 |

### 3.2 背单词模块

**功能描述：** 基于雅思高频词汇，提供系统化的单词学习体验。

| 功能项 | 说明 |
|--------|------|
| 词库来源 | 内置雅思核心词汇库（按难度分为基础 / 进阶 / 高频三个级别） |
| 单词卡片 | 每个单词展示：英文、音标、TTS 发音、中文释义、词性 |
| 跟读功能 | 播放标准发音，用户可跟读练习 |
| 例句展示 | 每个单词配备 1-2 个雅思场景常用例句（中英对照） |
| 学习进度 | 记录已学/未学/已掌握状态，支持按进度继续学习 |
| 收藏整合 | 与文章模块的收藏单词统一管理，可在此模块复习 |

### 3.3 练习模块

**功能描述：** 根据用户学习记录智能生成每日练习，巩固所学内容。

| 功能项 | 说明 |
|--------|------|
| 每日练习 | 系统根据当天学习内容自动生成练习题（选择题 / 填空题 / 拼写题） |
| 题目来源 | 基于用户已学单词 + 已读文章内容生成 |
| 错题记录 | 答错的题目自动记入错题本，标记错误次数 |
| 错题复练 | 下次生成练习时，自动将错题混入新题中重新考查 |
| 学习统计 | 展示每日练习正确率、连续打卡天数、累计学习单词数 |

## 4. 页面结构

```
首页（Dashboard）
├── 今日学习概览（已学单词数、练习正确率、连续打卡天数）
├── 快捷入口（继续学习 / 开始练习 / 导入文章）
│
├── 文章阅读页
│   ├── 文章列表（我的文章库）
│   └── 阅读详情页（播放、跟读、翻译、收藏）
│
├── 背单词页
│   ├── 词库选择（基础 / 进阶 / 高频）
│   ├── 学习卡片页（翻卡学习）
│   └── 生词本（收藏的单词统一管理）
│
├── 练习页
│   ├── 每日练习
│   ├── 错题本
│   └── 学习统计
│
└── 个人中心
    └── 学习记录与设置
```

## 5. 技术建议

| 层级 | 推荐方案 |
|------|---------|
| 前端 | React / Vue + TailwindCSS，响应式设计兼容移动端 |
| 语音播放 | Web Speech API（浏览器原生 TTS） |
| 翻译服务 | 接入第三方翻译 API（如 Google Translate / DeepL） |
| 数据存储 | 轻量方案：LocalStorage / IndexedDB（纯前端）；完整方案：后端 + 数据库 |
| 词库数据 | JSON 格式内置雅思词汇数据，支持后续扩展 |
| 练习生成 | 基于已学单词随机组题，错题按权重优先出现 |

## 6. MVP 范围（第一版）

优先实现以下功能，后续迭代扩展：

1. **文章粘贴导入** + TTS 播放 + 逐句跟读 + 中文翻译 + 单词收藏
2. **雅思基础词库** + 单词卡片学习 + 发音跟读 + 例句
3. **每日练习生成**（选择题为主） + 错题记录与复练
4. **纯前端实现**，数据存储在浏览器本地（降低开发复杂度）

## 7. 后续扩展方向

- 口语录音与发音评分
- AI 写作批改
- 听力专项训练（真题音频）

- 学习计划与备考倒计时
